FormChippy InputActive Example

This example demonstrates the new InputActive module which adds active state classes to inputs when they're checked/selected.

Standard Radio Buttons

The checked radio will receive the default active class.

Custom Active Class

These radio buttons use a custom active class.

Checkboxes

Multiple checkboxes can be active at once.

Input Containers

Using data-fc-input-container to define the active element.

Option with container

This option uses a container to define the element that receives the active class.

Another option with container

The container makes it easier to style complex input layouts.
